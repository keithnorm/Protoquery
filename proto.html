Making $ work for Prototype and jQuery

The difference between Prototype's $ method and jQuery's is that jQuery's takes a full css selector whereas Prototype's accepts a id as a string without any CSS selector syntax. So, in order to do something like:

$('#foo').show();

and have it work with either library, you need to extend Prototype's $ method like this:

$ = (function(old$) {
  return function(element) {
    element = typeof element == 'string'
      ? element.replace('#', '') 
      : element;
    return old$(element);
  }
})($);
